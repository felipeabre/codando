Bem-vindo ao Tutorial de HTML

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar a estrutura e o conteúdo de uma página na web.

Elementos Básicos

Existem vários elementos HTML que você pode usar para criar sua página web. Vamos começar com alguns elementos básicos:

  • <h1> - Define um cabeçalho de nível 1.
  • <p> - Define um parágrafo.
  • <a> - Define um link.
  • <img> - Insere uma imagem.
  • <ul> e <li> - Cria uma lista não ordenada.
  • <ol> e <li> - Cria uma lista ordenada.

Exemplos de Uso

Aqui estão alguns exemplos de como usar esses elementos:

Cabeçalhos

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Outro Subtítulo</h3>

Parágrafos

<p>Este é um parágrafo.</p>
<p>Este é outro parágrafo.</p>

Listas

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Exemplo Básico

Aqui está um exemplo de como você pode criar uma página HTML simples: 

            
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Bem-vindo!</h1>
    <p>Esta é minha primeira página HTML.</p>
</body>
</html>

Espero que este tutorial tenha sido útil para você começar a aprender HTML. Divirta-se criando suas próprias páginas web!
